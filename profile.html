Projects

Employee Attrition Using Machine Learning (Nov 2023 - Apr 2024)

Overview: This project focuses on predicting employee attrition using machine learning techniques. The goal is to help organizations proactively manage employee retention by understanding the factors that lead to attrition and predicting which employees are at risk of leaving.

Key Features:

  • Data Analysis: Historical employee data, including job satisfaction, performance, work-life balance, and tenure, is analyzed to identify key factors contributing to attrition.
  • Feature Engineering: Key features such as age, department, salary, and satisfaction levels are processed and used as inputs for the machine learning models.
  • Modeling: Machine learning algorithms like Logistic Regression, Random Forest, and Support Vector Machines (SVM) are used to predict employee attrition.
  • Evaluation: Models are evaluated using accuracy, precision, recall, and F1-score to ensure the most effective prediction.
  • Model Updating: The model is updated regularly with new employee data to improve prediction accuracy.

Technologies Used:

  • Python: For data analysis and machine learning modeling.
  • Pandas: For data manipulation.
  • Scikit-learn: To implement machine learning algorithms.
  • Matplotlib & Seaborn: For data visualization.

Impact:

This project helps companies predict which employees are at risk of leaving, allowing them to take action to retain valuable talent. By identifying key patterns in employee behavior, the model contributes to reducing turnover rates and improving employee satisfaction.

IOT Garbage Monitoring with Weight Sensing (May 2023 - Nov 2023)

Overview: This project is an IoT-based smart waste management system that monitors the fill levels and weight of garbage bins in real time. The system optimizes waste collection routes to reduce operational costs and promote environmental sustainability.

Key Features:

  • Weight Sensing: Each bin is equipped with a weight sensor that monitors the fill level and transmits the data to a central server.
  • Real-Time Monitoring: The system monitors the bins in real time and triggers alerts when a bin is near full capacity.
  • Route Optimization: The system optimizes collection routes by identifying which bins need to be collected, reducing unnecessary trips.
  • Alerts and Notifications: Real-time notifications are sent to the waste collection team when bins need to be emptied.

Technologies Used:

  • Arduino: Used to interface with the weight sensors.
  • IoT Protocols: Communication between sensors and server using protocols like MQTT or HTTP.
  • Cloud Computing: For storing and processing the data collected from the sensors.
  • Python/Node.js: Used for back-end processing and route optimization.

Impact:

This project helps reduce the environmental impact and cost of waste collection by optimizing collection routes and preventing unnecessary trips. The system also improves the efficiency of waste management services, ensuring bins are emptied before they overflow.
